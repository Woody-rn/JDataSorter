# Утилита фильтрации данных
## Описание
Программа для чтения данных из текстовых файлов (целые числа, вещественные числа, строки) и их сортировки по разным выходным файлам.  
Поддерживает сбор статистики, настраиваемые пути вывода, так же возможность добавления данных в существующие файлы.  
По умолчанию файлы создаются в текущей директории с именами:

* integers.txt - содержит все целые числа из входных файлов
* floats.txt - содержит все вещественные числа из входных файлов
* strings.txt - содержит все строки из входных файлов  

Важно!
> * Все файлы создаются в текущей директории, если не указаны пути вывода.  
> * Строки состоящие из символов пробела считаются строкой
> * Числа содержащие символы пробела является строкой
> * При указании каталога выходных файлов путь `/some/output` считается абсолютным путем  
>   `some/output` - относительным
> 

## Технологии
* Java 21  
* Gradle 8.8 - система сборки  
* Picocli 4.7.7 - фреймворк для парсинга командной строки

## Сборка проекта
1. Клонирование репозитория `git clone https://github.com/Woody-rn/JDataSorter.git`  
2. Сборка fat JAR со всеми зависимостями
`./gradlew shadowJar`
3. Запуск программы из командной строки
`java -jar JDataSorter-1.0-SNAPSHOT-all.jar <input_file>..`

## Доступные команды запуска
* `-o <путь>` задает каталог для выходных файлов	`-o /some/output`  
* `-p <префикс>` Задает префикс для имен выходных файлов	`-p sample- `   
  Символы `\/:*?"<>|` не допускаются в имени файла
* `-a`	Режим добавления в существующие файлы (по умолчанию перезапись файлов)
* `-s`	Краткая статистика (только количество элементов)
* `-f`	Полная статистика (мин/макс/сумма/среднее для чисел, мин/макс длина для строк)



## Примеры запуска

- Базовое использование  
`java -jar JDataSorter-1.0-SNAPSHOT-all.jar input1.txt input2.txt`  


- С префиксом и краткой статистикой  
`java -jar JDataSorter-1.0-SNAPSHOT-all.jar -s -p result_ in1.txt in2.txt`


- С указанием пути и режимом добавления  
`java -jar JDataSorter-1.0-SNAPSHOT-all.jar -a -o /output/path -f data.txt`


- Смешанный пример  
`java -jar JDataSorter-1.0-SNAPSHOT-all.jar -s -a -p sample- in1.txt in2.txt`
